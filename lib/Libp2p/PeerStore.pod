=head1 NAME

Libp2p::PeerStore - Registry for peer information

=head1 SYNOPSIS

    my $peer_store = Libp2p::PeerStore->new();

    $peer_store->add_addr( $peer_id, '/ip4/1.2.3.4/tcp/4001' );
    $peer_store->add_protocol( $peer_id, '/libp2p/autonat/1.0.0' );

    my $addrs = $peer_store->get_addrs( $peer_id );

=head1 DESCRIPTION

The PeerStore is a central location for storing information about peers we have encountered on the network.

It tracks addresses, protocols, metadata as arbitrary key/value pairs.

=head1 METHODS

=head2 C<add_addr( $peer_id, $multiaddr )>

Adds a new multiaddr for the given peer ID.

=head2 C<add_protocol( $peer_id, $protocol )>

Registers a supported protocol for the given peer ID.

=head2 C<get_addrs( $peer_id )>

Returns an arrayref of multiaddrs for the peer.

=head2 C<get_protocols( $peer_id )>

Returns an arrayref of supported protocols for the peer.

=head2 C<peers_supporting( $protocol )>

Returns an arrayref of peer IDs that support the given protocol.

=head2 C<set_metadata( $peer_id, $key, $val )>

Sets a metadata value for the peer.

=head2 C<get_metadata( $peer_id, $key )>

Returns a metadata value for the peer.
